popifyElement

Like a class, gives selected element abiltiy to create, trigger, mutate, and animate different popups

popifyElement(selector: string): Object
Parameters
selector (string) A css selector for an element that we want to trigger popups with
Returns
Object: An object that has access to the rest of the functions of the library. Allowing us to create and customize popups for the seleced element.

createPopUp

Accessed from the object returned by popifyElement, creates a popup initially linked to the selected trigger element

createPopUp(params: Object): HTMLElement
Parameters
params (Object) The different properties of the popup
Name Description
params.type String The type of popup, must be either "image" "text" or "custom"
params.content (String | HTMLElement) The popup content, either an image link, text, or an html node depending on the type value
params.height Integer The height of the popup in px
params.width Integer The width of the popup in px
params.persist Boolean (default false) The popup needs to be manually clicked to close
params.displayInitially Boolean (default false) Show the popup when the page first loads
params.draggable Boolean (default false) If the popup can be moved, requires persist=true
params.resizable Boolean (default false) If the popup can be resized, requires persist=true
params.textOptions Object (default null) Options for a text popup
params.textOptions.fontFamily String (default null) Text font family
params.textOptions.backgroundColor String (default 'white') Popup background colour
params.textOptions.padding Integer (default null) Padding element containing text
params.textOptions.fontSize Integer (default null) Text font size
params.expand Object (default null) How much to expand the popup when a mouse hovers over it, requires persist=true (without animation)
params.expand.newHeight Integer Required if expand parameter is passed in. New height the popup will expand to
params.expand.newWidth Integer Required if expand parameter is passed in. New width the popup will expand to
params.shift Object (default null) How much to shift the popup relative to appearing right below the element that triggers it.
params.shift.x Integer Required if shift parameter is passed in. How many pixels to shift right
params.shift.y Integer Required if shift parameter is passed in. How many pixels to shift down
Returns
HTMLElement: Returns the popup DOM element ex <img src=' www.popupimage '... />

setAbilities

Set or change the attributes of a particular popup

setAbilities(popup: HTMLElement, params: Object)
Parameters
popup (HTMLElement) The popup element
params (Object) we may change whether it only shows on hover, if it is draggable, resizable, and if it expands
Name Description
params.persist Boolean (default false) The popup needs to be manually clicked to close
params.draggable Boolean (default false) If the popup can be moved, requires persist=true
params.resizable Boolean (default false) If the popup can be resized, requires persist=true
params.expand Boolean (default false) The popup needs to be manually clicked to close
params.expand.newHeight Integer Required if expand parameter is passed in. New height the popup will expand to
params.expand.newWidth Integer Required if expand parameter is passed in. New width the popup will expand to
params.expand Object (default null) How much to expand the popup when a mouse hovers over it, requires persist=true

animate

Add animations to a popup

animate(popup: HTMLElement, params: Object)
Parameters
popup (HTMLElement) The popup element
params (Object) Different animations we can add to the popups
Name Description
params.float Object (default null) Whether the popup bobbles up and down
params.float.range Integer Required if float parameter is passed in. How many pixels up and down does the popup move
params.float.speed Integer Required if float parameter is passed in. How long in miliseconds does it take for the popup to cover the range
