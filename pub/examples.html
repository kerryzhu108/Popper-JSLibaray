Recognizing Different Species of Mushrooms API

This library is all about encapsulation. For this demonstration, the theme will be a webpage that's all about mushrooms. While this is quite sepecific, I think the way it uses the library's functionalities is a good representation of the how the library will be used in general.

Basic Popup

Psilocybe quebecensis is a moderately active mushroom in the section Aztecorum, having psilocybin and psilocin as active compounds. Hover over me to take a look at this not so extraordinary shroom.

Custom locations, resiability, and animatons

You may have noticed that the previous mushrooms spawned at unusal places, this can be set using the shift parameter. For example, we can make this picture of Psilocybe quebecensis appear at the top of the page near releveant information! We can also expand it by clicking that little button on the bottom right, moving the curosor, and clicking again to drop. We have also animated it to fade and float using the animate method.

Multiple popups

Hover over the mushroom stew to see multiple popups appear around it, useful for viewing its ingredients or just to see more mushrooms. Sample code (resizable).

Text popups

We can also create text popups where we get to specify the background and font. Developers can either make the text fit inside the box or have it so that users can resize the text popup to read more. They can also have it so when a user clicks an element and then hovers over its popup, it automatically expands to reveals more text. Code sample

Changing attributes later

The developer is also able to change the functionalities of already created popups later. This mushroom is not draggable. Clicking this will now make it draggable! This is done though the .setAbilities method.

Custumoizable elements

Finally we can pass custom html elements into our popups for even more customization. This example has popups that trigger more popups. To do this we pass in our own html element to .createPopup and for each button inside it, we call .createPopup on it.
